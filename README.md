# kousien-extension README

スモウルビー甲子園のメソッドを補完してくれる拡張機能です。

対応しているメソッドは以下の通りです。

- `move_to`
- `calc_route`
- `get_map_area`
- `set_dynamite`
- `map`
- `map_all`
- `locate_object`
- `other_player_x`
- `other_player_y`
- `enemy_x`
- `enemy_y`
- `goal_x`
- `goal_y`
- `player_x`
- `player_y`
- `set_name`
- `connect_game`
- `turn_over`
